# Modüler Masraf Enterprise v20.1

Bu sürüm yalnızca temel altyapı ve giriş sistemidir.

## GitHub
ZIP'i bilgisayarınızda açın. ZIP'in içindeki dosya ve klasörleri boş GitHub reposunun köküne yükleyin.

## Railway
1. GitHub reposunu yeni Railway projesine bağlayın.
2. Aynı Railway projesine yeni PostgreSQL ekleyin.
3. Uygulama servisinde `DATABASE_URL` değişkenini yeni PostgreSQL'in `DATABASE_URL` değişkenine referans olarak bağlayın.
4. `SESSION_SECRET` ve `NODE_ENV=production` ekleyin.
5. Deploy edin.
